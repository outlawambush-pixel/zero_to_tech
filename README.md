# Zero To Tech

一个面向初学者的前端入门小项目，用于练习 HTML、CSS 和 JavaScript 的基础用法。

## 项目说明

这个项目的目标是把“从零开始接触技术”落成一个可运行的小网页，通过简单直观的方式理解页面结构、样式和交互。

## 项目结构

```
zero_to_tech/
├── super.html    # 主页面
├── style.css     # 页面样式
├── script.js     # 页面交互逻辑
└── README.md     # 项目说明
```

## 文件职责

- `super.html`：页面骨架，包含标题、正文、按钮和脚本引入。
- `style.css`：负责页面布局、颜色、按钮样式和悬停效果。
- `script.js`：处理页面中的交互行为，例如点击按钮后的内容变化。

## 快速开始

直接用浏览器打开 `super.html` 即可看到效果：

```bash
# Windows
start super.html

# macOS
open super.html
```

也可以将 `super.html` 拖入浏览器中打开。

## 学习建议

1. 先阅读 `super.html`，理解标签和结构。
2. 修改 `style.css`，观察颜色、间距和布局变化。
3. 查看 `script.js`，尝试理解事件和函数如何配合。
4. 逐步加入新的 HTML 元素或交互效果。
